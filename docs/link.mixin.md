## [@insite](../../README.md)/[sass-mixins](../README.md)/link.mixin

Faire un lien et ses différents états (:link, :visited, :hover, :focus, :active)

* *$link: $c-primary*
* *$hover: $c-secondary*
* *$deco: underline*
* *$deco--hover: none*

#

#### Usage :


```bash
@include a-link(#000, #333, none, underline);
```
