## [@insite](../../README.md)/[sass-mixins](../README.md)/transform.mixin

Syntaxe raccourcie de la propriété transform

* *$translateX: 0*
* *$translateY: 0*
* *$rotate: 0deg*
* *$scale: 1*

#

#### Usage :


```bash
@include transform(20px);
```

ou

```bash
@include transform(0 0, 90deg);
```

ou encore

```bash
@include transform(0 0, 0, .5);
```
